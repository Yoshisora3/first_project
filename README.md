# first_project
Webで簡単に使用できるメモ帳です  
色々な技術を学習するために試行錯誤しながら作成しました

## 主な機能
- メモの一覧表示・追加・削除・更新
- ログイン機能
## 対応プラットフォーム
- Windows

## 使用技術
・Python
・Flask
・HTML
・CSS
・SQLite
・SQLAlchemy
・Jinja2
・Flask-Login

## 今後の機能改善  
- パスワードのハッシュ化  
- ユーザーごとにメモを管理
- UIの改善

## 使用方法  
1　Python、Flask、SQLAlchemy、flask-loginをインストール  

2　Python app.pyで起動します
