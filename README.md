# first_project
Webで簡単に使用できるメモ帳を作成しました
## 主な機能
- メモの追加・削除・更新
- ログイン機能（試作）
## 対応プラットフォーム
- Windows

## 使用技術
・Python
・Flask
・HTML
・CSS
・SQLite
・SQLAlchemy
・Jinja2
・Flask-Login

## 機能改善  
メモの改行ができるように修正する予定です  
ログイン機能を改善予定です

## 使用方法  
1　Python、Flask、SQLAlchemy、flask-loginをインストール  

2　Python app.pyで起動します

## メモ帳機能の説明(作成中です)
### メモの追加
追加したいメモを追加するのテキストエリアに入力して右にある追加のボタンをクリック  

メモ一覧に追加したメモが表示されます
